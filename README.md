# getrak-dsp
Getrak Design System Package

## Como posso usar o pacote?
Faça o download do projeto na sua máquina, deixe o pacote ao lado da pasta do seu projeto.
Agora que você tem o `getrak-dsp`, você deve instalar o [Adobe XD extenstion](https://marketplace.visualstudio.com/items?itemName=Adobe.xd&ssr=false#overview) no seu ambiente VS Code para abrir qualquer pacote dsp. Depois de instalar a extensão, abra o painel do Adobe XD e selecione a opção `Load package`. Então navegue à pasta que se encontra o `getrak-dsp` (DSPs contem um arquivo dsp.json no nível raiz) e selecione-a. Pule as etapas de cópia de arquivos e pronto! Você já pode fazer uso do Getrak Design System :D

Se possível, assista uma pequena [lista de vídeos](https://letsxd.com/vscode) explicando como consumir um DSP package.
