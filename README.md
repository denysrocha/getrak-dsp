# getrak-dsp
Getrak Design System Package

## Instalação
Para instalar o Getrak Design System Package no seu projeto é só executar o comando `npm install getrak-dsp`

## Como posso usar o pacote?
Primeiro você deve instalar o [Adobe XD extenstion](https://marketplace.visualstudio.com/items?itemName=Adobe.xd&ssr=false#overview) no seu ambiente VS Code para abrir qualquer pacote dsp. Depois de instalar o getrak-dsp, abra o painel do Adobe XD e selecione a opção `Load package`. Então navegue à pasta `node_modules` do seu projeto, e selecione a pasta `getrak-dsp` (DSPs contem um arquivo dsp.json no nível raiz). Se possível, assista uma pequena [lista de vídeos](https://letsxd.com/vscode) explicando como consumir um DSP package.
